$(a + b)^n = ?$  
$(1 + x)^0 = 1$  
$(1 + x)^1 = 1 + x$  
$(1 + x)^2 = 1 + 2x + x^2$  
$(1 + x)^3 = 1 + 3x + 3x^2 + x^3$  
$(1 + x)^4 = 1 + 4x + 6x^2 + 4x^3 + x^4$

Each row $n$ consist of the $\binom{n}{k}$:  
$\binom{4}{0} = 1$, $\binom{4}{1} = 4$, $\binom{4}{2} = 6$, $\binom{4}{3} = 4$, $\binom{4}{4} = 1$.

# $Tri = 〈r , f 〉$
![[Pasted image 20250510171922.png]]

Equivalences of actions like the following are called [[Relations]]:

$$ r^3 = 1, \quad f^2 = 1, \quad rf = fr^2, \quad fr = r^2f. $$

| Element | Inverse |
|---------|---------|
| $1$     | $1$     |
| $r$     | $r$     |
| $r^2$   | $r$     |
| $f$     | $f$     |
| $rf$    | $rf$    |
| $r^2f$  | $r^2f$  |

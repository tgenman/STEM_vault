> [!tip] The relative complement of set ${B}$ with respect to set ${A}$ 
is the difference of sets ${A \setminus B}$, when ${B \subseteq A}$.
Such complement is denoted as ${B_A}$.
In this case, the set ${B \setminus A}$ is empty.

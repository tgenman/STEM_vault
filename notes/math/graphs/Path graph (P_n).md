> [!tip] Path $P_n$
> is a [[Simple Graph (G)|graph]] of order $n$ and size $n−1$ whose vertices can be labeled by $v_1,v_2,...,v_n$ and whose edges are $vi, v_{i+1}$ for $i = 1,2,...,n−1$ (for an integer $n≥1$)

Let $k \leq n$  and $A = \set{a_1,..., a_n}$ be a finite set

A ==k-combination== from n elements (n-element set A) is an k-element subset of A. 
No Order
No repetition:
$$
C_{n, k} = \binom{n}{k} = \frac{n!}{k!(n-k)!}
$$
With repetition:

8! / (64!/(8!*56!))
